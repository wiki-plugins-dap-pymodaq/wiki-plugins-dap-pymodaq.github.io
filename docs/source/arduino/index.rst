Arduino plugin
==============

``pymodaq_plugins_arduino`` groups a set of instruments built around an **Arduino** or
**ESP32** board. Some instruments use the `Telemetrix <https://mryslab.github.io/telemetrix/>`_
library (Arduino over USB); others use Telemetrix-AIO to talk to an ESP32 over WiFi.

.. note::

   Detailed instrument documentation (actuators, viewers, the ColorSynthesizer
   extension and installation) is added in a later sprint.
