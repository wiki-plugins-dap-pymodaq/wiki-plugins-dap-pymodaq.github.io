Wiki Plugins DAP Pymodaq

La documentation technique des trois plugins PyMODAQ du projet.

Le projet porte le nom de code abrégé « DAP Pymodaq » — c'est ce nom court qui apparaît dans la barre de gauche. Son but : migrer les bancs d'expérimentation thermique de LabVIEW vers PyMODAQ, un framework d'acquisition de données libre et open source.

BTS CIEL · Session 2026 Partenaire : CETHIL Lycée Edouard Branly, Lyon

Le projet en bref

🏫 Établissement

Lycée Edouard Branly, Lyon.

🤝 Partenaire

CETHIL — Centre d'énergétique et de thermique de Lyon (CNRS, INSA Lyon, UCBL 1).

🎯 Objectif

Remplacer LabVIEW par PyMODAQ sur les bancs thermiques et concevoir un nouveau plugin Raspberry Pi.

Technologies & exigences techniques
Catégorie Éléments Clés Source / Exigence
Framework DAQ PyMODAQ Respecter les règles et chartes de développement PyMoDAQ.
Langage Python Utilisation exclusive du langage Python.
Matériel RPi (Zero/3), Arduino Nouveau module/plugin à concevoir pour le Raspberry Pi.
Communication ZeroMQ S'appuyer sur ZeroMQ (messagerie) pour la com avec les RPi.
Gestion Git, GitLab Méthode Agile avec sprints (min 3 versions : V1, V2, V3).
Environnement PyCharm, Win/Lin Les applications devront être testées pour Windows et Linux.
Exportation HDF5 (.h5) Assurer l'exportation des données de mesure.
Organisation de l'équipe
E1

E2

E3

E4
