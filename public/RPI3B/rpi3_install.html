Installation et Configuration du Raspberry Pi 3

Cette section décrit l'installation et la configuration du système d'exploitation Raspberry Pi OS sur le Raspberry Pi. La configuration inclut la mise en place d'une connexion SSH, la définition éventuelle d'une adresse IP statique et l'activation du service I2C.

1. Installation du système d'exploitation Raspberry Pi OS

Pour l'installation du système d'exploitation Raspberry Pi OS, la première étape consiste à flasher l'image du système d'exploitation sur la carte micro SD en utilisant un autre ordinateur (Windows, Mac ou Linux).

Pour graver l'image du système d'exploitation Raspberry Pi OS, nous utiliserons le logiciel officiel de Raspberry appelé Raspberry Pi Imager. Vous pouvez le télécharger et l'installer à partir du site suivant : Lien vers la page d'installation Raspberry PI OS.

Une fois le logiciel Raspberry Pi Imager installé sur votre PC, suivez les étapes de configuration demandées.

Attention : Si vous prévoyez une connexion ultérieure via SSH, il est crucial :
  • De noter le nom d'utilisateur et le nom de la machine (hostname).
  • D'activer la connexion SSH durant la configuration.
Cela vous permettra de vous connecter à distance à votre machine Raspberry Pi.

2. Connexion via SSH

Une fois la carte SD préparée et insérée dans votre Raspberry Pi, allumez l'appareil et connectez-le soit à votre réseau local, soit directement à votre PC.

Vous pouvez établir une connexion à distance avec votre Raspberry Pi via SSH, directement depuis PowerShell.

Vous avez deux options pour vous connecter :

# Si vous connaissez l'adresse IP de votre machine :
ssh votre_nom_d'utilisateur@adresse_ip_de_machine
# Si vous ne connaissez pas l'adresse IP de votre machine :
ssh votre_nom_d'utilisateur@nom_de_votre_machine

Après avoir entré la commande, il vous sera demandé de saisir le mot de passe de votre session pour finaliser la connexion.

3. Configuration de l’OS

Dès la première connexion, il est fortement conseillé de mettre à jour votre système afin de prévenir d'éventuelles erreurs. Pour ce faire, utilisez la commande suivante :

sudo apt update && sudo apt upgrade -y

De plus, si vous le souhaitez, vous avez la possibilité de modifier l'adresse IP de votre Raspberry Pi pour lui en attribuer une statique via l'outil nmtui. Il est également possible de changer le nom d'hôte de votre machine avec cet outil.

sudo nmtui

4. Configuration des broches GPIO (Pigpio)

Pour utiliser les broches GPIO, vous devez d'abord installer le paquet pigpio à l'aide de la commande :

sudo apt install pigpio

Pour automatiser le lancement de ce paquet, vous devez ensuite utiliser les commandes :

sudo systemctl enable pigpiod
sudo systemctl start pigpiod

5. Activation du service I2C

Pour activer le service I2C, nécessaire à la communication avec les composants externes du Raspberry Pi, veuillez utiliser la commande :

sudo raspi-config

Dans ce menu, vous devez sélectionner l'option Interface Options, puis choisir l'option I2C. Vous n'avez plus qu'à accepter pour activer le service.
